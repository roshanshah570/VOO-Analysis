# VOO ETF Performance Analysis

Analysis of VOO ETF performance over one year using R. Examines quarterly 
trends in gains, trading volume, and price volatility.

## Key Findings
- VOO experienced more Gain days than No Gain days across all quarters
- Q2 showed the highest volatility, trading volume, and number of daily gains
- Strong positive correlation between volume and daily price range

## Tools Used
R, ggplot2, dplyr, lubridate

## Data Source
Yahoo Finance — VOO historical data (Nov 2024 – Nov 2025)
